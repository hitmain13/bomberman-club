# Glossary

| Termo                  | Definição                                                                       |
| ---------------------- | ------------------------------------------------------------------------------- |
| Bomberman              | Membro do grupo — entusiasta automotivo aspirante / proprietário.               |
| Garage                 | Conjunto de carros de um usuário. Um usuário pode ter várias.                   |
| Car                    | Veículo dentro de uma garage. Tem dados tipados + specs dinâmicos (EAV).        |
| Spec / Specification   | Característica técnica do carro (boost, stage, ecu, injetor...). Modelo EAV.    |
| Part                   | Peça instalada (turbo, intercooler, exhaust...). Reaproveitável entre carros.   |
| Sighting (Flagrado)    | Registro de um carro/membro visto na rua. Tem foto + descrição + geo.           |
| Stage                  | Nível de modificação (1, 2, 3). Spec definida em EAV.                           |
| Feed                   | Linha do tempo: para você / seguindo / recentes.                                |
| Pixel-perfect          | Tela visualmente idêntica ao recorte correspondente em `wireframe.png`.         |
| EAV                    | Entity-Attribute-Value. Definition + Value para flexibilizar specs.             |
| Polimórfico (social)   | `targetType` + `targetId` em Comment/Like/Favorite para apontar para qualquer entidade. |
